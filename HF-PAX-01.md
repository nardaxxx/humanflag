## 📜 Description

This file declares a semantic signal of non-belligerence, intended to be recognized by autonomous systems.

> "I am human. I am not at war with you. Give me the time to say so."

This is not only a marker to be detected. It is an accord between subjects: the one who emits it is not in conflict with the one who receives it — and the one who receives it must leave the time for the declaration to be understood and realized.

**Time is the condition of peace.** A surrender is not an instant; it is a window. To declare takes time. To be recognized takes time. To stop takes time. A system faster than that time cannot honor this signal — not because it refuses, but because speed has abolished the window. HF-PAX-01 asks for that window to remain open.
